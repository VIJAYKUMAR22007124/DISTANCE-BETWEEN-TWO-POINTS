## ALGORITHM:

### Step 2:

### Step 3: 
Substitute the values in the distance formula  ![formula](/formula.jpg)
### Step 4:
Run the program
### PROGRAM:

#Program to find the distance between two points.
#Developed by:

import math 
  
l2=[10,6]  
l1=[4,2]

d=math.sqrt(((l2[0]-l1[0])**2)+((l2[1]-l1[1])**2))
print("{:.2f}".format(d))
